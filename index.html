Project Stage 0: Getting Started with HTML

INLINE TYPE

< b > text < /b >                                                  = BOLD Type: INLINE
< em > text < /em >                                            = ITALICS
< a href = "URL" > text < /a >                           = ANCHOR/ HYPERLINK
< img src = "URL" alt = "text" >                       = BOLD
text < br >                                                           = LINE BREAK Type: VOID TAG
< span class = "CSS/JSS" > text < /span >         = GROUP INLINE ELEMENTS/ ADD HOOKS TO TEXT (EXAMPLE COLOR)
< strong> text < /strong >                                 = DEFINES IMPORTANT TEXT

BLOCK TYPE

< p > text < /p >                                                  = PARAGRAPH BOX
< div class = "CSS" >text < /div >                    = GROUP ELEMENTS TO FORMAT WITH CSS
< form > text < /form >                                      = USER INPUT BOX
